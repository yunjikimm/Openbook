## Openbook
개발자 기술 면접 준비를 위한 오픈북 iOS 앱

- 팀원
    - 1인 개발
- 진행 기간
    - 개발: 2024.05.15 ~ 2024.05.27
<br/>

## 개발 환경 및 기술
- 개발 환경
    - Swift 5, xcode 15.0
    - 라이트모드 지원(다크모드 미지원), 가로모드 미지원
- Deployment Target
    - iOS 15.0
- 라이브러리
    - Firebase
<br/>

## 아키텍처
MVVM + Clean Architecture

<br/>
